# 纯css实现导航栏动画

# 控制动画进出方向

一般情况下的hover展示：左进左出

使用scale 与 transfrom-origin控制动画方向：左进右出

实现控制动画方向的关键点：使得 hover 动画的进入与离开产生两种不一样的效果 。也就是采用 transform: scale() 以及 transform-origin来实现。

transform: scale() 实现线条运动

CSS 函数 scale() 用于修改元素的大小。可以通过向量形式定义的缩放值来放大或缩小元素，同时可以在不同的方向设置不同的缩放值。

所以这里使用 transform: scaleX(0) 与 transform: scaleX(1) 来改变线条的显示与隐藏。

transform-origin 实现线条运动方向

transform-origin 属性允许改变被转换元素的位置。

2D 转换元素能够改变元素 x 和 y 轴。3D 转换元素还能改变其 Z 轴。

为了更好地理解 transform-origin 属性，可以查看这个例子

CSS代码如下

.scale-hover::before {
    position: absolute;
    bottom: 0;
    left: 0;

    width: 90px;
    height: 2px;

    background: rgb(233, 165, 200);

    transition: transform .5s;
    transform: scale(0);
    transform-origin: 100% 0;

    content: "";
}
.scale-hover:hover::before {
    transition-timing-function: linear;
    transform: scale(1);
    transform-origin: 0 0;
}

注意点

  • 可以尝试使用transition-timing-function 缓动函数，可以让动画更加流畅具有美感；

  • 线条的 transition 设置的是 transition: transform .5s 而不是 transition: all .5s，两种效果会有所不同。

# 导航栏下划线跟随效果

实现方式

1、首先借助伪元素给每一个li标签加上下划线；

2、利用绝对定位，将li的伪元素的宽度设置为0，在hover的时候，宽度从0到100%变化；

3、左移左出，右移右出；当从第一个 li 切换到下一个 li 的时候，将下划线的初始位置设置为left：100%

4、使用~ 选择符，不改变当前 hover 的 li 的下划线移动方式却能改变它下一个 li 的下划线的移动方式；

~也叫做相邻同级连续选择符： A~B 表示匹配A之后的B元素，直到出现另一个A元素。

未加~ 选择符之前的效果

完整的CSS代码如下

.list::before {
    position: absolute;
    top: 0;
    left: 100%;

    border-bottom: 2px solid rgb(233, 180, 189);
    width: 0;
    height: 100%;

    transition: .2s all linear;

    content: "";
}
.list:hover::before {
    top: 0;
    left: 0;
    z-index: -1;

    border-bottom-color: rgb(236, 184, 192);
    width: 100%;

    transition-delay: .1s;
}
.list:hover ~ .list::before {
    left: 0;
}
